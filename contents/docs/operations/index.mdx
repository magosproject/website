The Operations section is what you read when Magos is past Quickstart and into a production cluster.

- [Authentication](/docs/operations/authentication): configure the admin account and OIDC for the Magos UI and API. Includes when to disable the admin account and how to manage credentials.
- [Security and RBAC](/docs/operations/security): the runner-pod isolation model, ServiceAccount conventions, secret handling, and cluster permissions.
- [Observability](/docs/operations/observability): Prometheus metrics, structured run logs, and the data behind the UI.
